Known Data
==========

Known data is a database containing names and related V\ :sub:`d` \ values of various drugs. It is also an .csv file.

.. csv-table::
    :header: File type, Name Data, :math:`V_d` Data

    .csv, Name, V / V_d

.. caution::
    Entertaining other headings will not work.
